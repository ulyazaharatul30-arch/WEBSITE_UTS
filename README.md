# K-Drama Lovers ❤️

A modern, cinematic K-Drama discovery website built with HTML, CSS, JavaScript, and Bootstrap 5.

## 🎬 Project Structure

```
/k-drama-lovers
│
├── index.html               ← Homepage with hero carousel & genre sections
├── movies.html              ← All dramas with search & filter
├── about.html               ← About Us + founder profile
├── contact.html             ← Contact form
├── css/
│   └── style.css            ← Main stylesheet
├── js/
│   └── script.js            ← Main JavaScript
├── pages/
│   ├── mouse.html
│   ├── boyfriend-on-demand.html
│   ├── all-of-us-are-dead.html
│   ├── the-manipulated.html
│   ├── spirit-fingers.html
│   └── a-shop-for-killers.html
├── assets/
│   ├── images/
│   ├── icons/
│   └── videos/
└── README.md
```

## 🎨 Design System

**Color Palette:**
- Coral/Orange: `#E8845A`, `#F0A06A`
- Peach: `#F5C09A`
- Deep Red: `#8B2252`
- Dark Purple: `#4A1A5C`
- Navy Card: `#1C2B4A`

**Fonts:**
- Display: Playfair Display (headings)
- Body: Nunito (text)

## 🚀 Features

- ✅ Loading animation screen
- ✅ Auto-playing hero carousel
- ✅ Live search (homepage + movies page)
- ✅ Genre filter tabs with URL param support
- ✅ Hover card animations
- ✅ Scroll reveal animations
- ✅ Scroll-to-top button
- ✅ Active navbar state detection
- ✅ Embedded YouTube trailers
- ✅ Star ratings
- ✅ Similar dramas section
- ✅ Glassmorphism founder card
- ✅ Working contact form with success feedback
- ✅ Fully responsive (mobile/tablet/desktop)
- ✅ Bootstrap Icons
- ✅ Google Fonts

## 🛠️ Tech Stack

- HTML5 (semantic structure)
- CSS3 (custom properties, animations, glassmorphism)
- JavaScript ES6+ (IntersectionObserver, URLSearchParams)
- Bootstrap 5.3
- Bootstrap Icons 1.11
- Google Fonts (Playfair Display + Nunito)

## 📱 How to Run

Just open `index.html` in any modern browser. No build step required.
